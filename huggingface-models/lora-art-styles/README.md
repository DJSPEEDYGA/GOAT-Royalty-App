# HuggingFace LoRA Art Style Models

> Curated collection of LoRA (Low-Rank Adaptation) fine-tuned models for artistic style transfer and image generation. Compatible with FLUX, Stable Diffusion XL, and other diffusion model backends.

---

## 🎨 What are LoRA Art Models?

LoRA models are lightweight fine-tuned adapters that can be applied on top of base image generation models (like FLUX or SDXL) to produce images in specific artistic styles. They're small (typically 10-100MB), fast to load, and can be combined.

## 🚀 Usage

### With Diffusers (Python)

```python
from diffusers import FluxPipeline
import torch

# Load base FLUX model
pipe = FluxPipeline.from_pretrained(
    "black-forest-labs/FLUX.1-dev",
    torch_dtype=torch.bfloat16
)
pipe.to("cuda")

# Load a LoRA adapter
pipe.load_lora_weights("Niferpita/vangogh_style_LoRA")

# Generate in Van Gogh style
image = pipe(
    "A starry night over Ms Money Penny Store, vibrant swirling sky",
    num_inference_steps=30,
    guidance_scale=7.5,
).images[0]

image.save("vangogh_store.png")
```

### With ComfyUI

1. Download the LoRA `.safetensors` file from HuggingFace
2. Place in `ComfyUI/models/loras/`
3. Add a "Load LoRA" node in your workflow
4. Connect to your FLUX/SDXL checkpoint loader

### With Automatic1111 / Forge

1. Download the LoRA to `stable-diffusion-webui/models/Lora/`
2. Use `<lora:model_name:weight>` in your prompt
3. Adjust weight (0.5-1.0) for style intensity
