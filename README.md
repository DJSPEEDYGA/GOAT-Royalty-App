## 🎯 What Is GOAT Royalty?

GOAT Royalty is a **complete, all-in-one music business platform** built for independent artists, producers, publishers, and rights holders. It combines professional-grade music production tools, AI-powered business intelligence, royalty tracking, distribution management, and a full creative suite — all in a single web app that runs locally on any device.

Think of it as **Pro Tools + FL Studio + DistroKid + Spotify for Artists + Auto-Tune + ChatGPT** — unified under one roof, owned by you, running on your machine.

## 🧠 Technical Highlights

### Canvas 2D API
- DAW lane rendering with waveform clips and animated playhead
- Neural graph visualization — 11 nodes, orbital animation, live data packets
- Orchestral score notation — 5-line staves, clef symbols (𝄞/𝄢), note heads
- EQ curve rendering with spline interpolation
- VU meter animation with peak hold and RMS ballistics

### Web Audio API
- **Procedural drum synthesis** — kick (OscillatorNode + exponential ramp), snare (noise buffer + BiquadFilter), hi-hat (noise + highpass filter), 808 bass (sine + WaveShaper distortion)
- **Step sequencer scheduling** via `setInterval((60/bpm/4) * 1000)`
- Real-time audio processing pipeline with gain, EQ, and compression nodes

### requestAnimationFrame Animation Loops
- Studio playhead + lane waveform render
- Video preview with film grain + letterbox + timecode overlay
- Orchestral score scrolling system
- Neural graph orbital node animation

### Canvas Knob Engine
- `drawKnob(canvas, value, color, size)` — arc-based rotary knob renderer
- Mousedown/mousemove drag interaction system
- Used across SSL Mixer, Vocal Studio, Film Score, Beat Maker

### AI Agent System
- 11 specialized agents: Music Producer, Beat Maker, Mixing Engineer, Vocal Coach, Business Manager, Legal Advisor, Distribution Manager, Marketing Director, Film Composer, Fan Engagement, and Autopilot
- Keyword-based routing to best-match agent
- Canvas neural graph with orbital animation and animated data packets
- Autopilot mode with task cycling and progress simulation

## 🚀 Advanced Deployment Options

### Local Web Server (recommended for full features)
```bash
# Python 3 (built-in — no install needed)
cd GOAT-Royalty-App/web-app
python3 -m http.server 8080
# Open: http://localhost:8080

# Node.js serve
npx serve web-app
# Open: http://localhost:3000

# PHP
php -S localhost:8080
```
