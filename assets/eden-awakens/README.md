# Eden Awakens Asset Workspace

This folder separates the flattened Eden Awakens artwork into practical production buckets.

- `originals/`: untouched source PNG files copied from Downloads.
- `layer-candidates/`: files to open in Photoshop or AI tools when extracting text, title, foreground, or background plates.
- `app-backgrounds/`: stable filenames for GOAT/Lexi web, launcher, and touch-screen backgrounds.
- `reference/`: screenshots, prompt notes, or visual references.
- `exports/`: final resized/cropped outputs for apps, decks, and promos.

If a PSD, layered source file, or Stable Diffusion prompt/settings file is found later, put it in `reference/` first before making derived exports.
