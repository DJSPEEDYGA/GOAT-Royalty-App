🚀 One-Click Model Downloader

Download AI models DIRECTLY to your server — no terminal, no git, just click

📋 How It Works

This downloader runs git clone on your server (not your browser). Models download to /var/www/models/ and can be streamed from there.

⚠️ Requires backend: The downloader needs the backend script installed on your server. See install instructions below

⚙️ Install the Backend Downloader

Run this ONE command on your server to enable actual downloads:

curl -fsSL "https://raw.githubusercontent.com/DJSPEEDYGA/GOAT-Royalty-App/main/INSTALL-MODEL-DOWNLOADER.sh" | bash

After install, this page will have working download buttons. Models download to /var/www/goat/models/ and are accessible at http://your-server/models/MODEL_NAME/
