Quick reference for AAX SDK, Pro Tools Scripting API, and EUCON. Full docs in the downloaded SDK packages.
AAX SDK 2.9.0 — Key Classes
AAX_IEffectParameters
Core interface for all plugin parameter management. Override GetChunkSize, GetChunk, SetChunk for preset management.
AAX_CEffectGUI
Base class for building plugin UI. Use with VSTGUI or custom drawing. SetViewSize, Draw, HandleEvent.
AAX_CMonoEffectParameters
Simplified parameters class for mono/stereo effects. Most common starting point for audio effects.
AAX_IComponentDescriptor
Describes the plugin's audio processing component — channel configs, latency, bypass.
AAX_IPropertyMap
Set plugin properties: category, stem format, manufacturer/product IDs, page table.
AAX_CParameter
Typed parameter class. Use AAX_CLinearTaperDelegate for linear, AAX_CLogTaperDelegate for log scaling.
Pro Tools Scripting — Common Commands
transport/play
Start playback from current cursor position. No parameters required.
transport/stop
Stop playback or recording.
track/set_solo?name=X
Solo track by name. Add &state=true/false to toggle.
track/set_mute?name=X
Mute/unmute a specific track by name.
session/set_bpm?bpm=140
Set session tempo. Requires conductor ruler to be active.
bounce/to_disk?format=wav
Bounce session to disk. Params: format (wav/aiff/mp3), bit_depth, sample_rate.
