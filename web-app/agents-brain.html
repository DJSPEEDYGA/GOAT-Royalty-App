⚡ How It Works

Every agent sends your message to the /brain/agent/ endpoint on your Intel server. The brain tries engines in this order — whichever answers fastest wins:

1

🔒 Ollama (Local)

Runs on YOUR machine. Free, private, works offline. Install from ollama.com, then ollama pull gemma3:4b.

2

⚡ NVIDIA Build

Free cloud tier. Llama 3.3 70B on NVIDIA GPUs. Key already set in goat-app/.env.

3

🌟 Gemini 2.5 Flash

Google's free tier. Fastest response. Your key is set. Auto-upgrades to Gemini 2.5 Pro for "reason" tasks.

4

🚫 OpenAI: REMOVED

Deleted from the stack. You don't pay them a dime. Your local Gemma IS your GPT.

🤖 How to add your own agent: Open goat-intel-server/goat_intel.py, find AGENT_PERSONAS, and add a new entry: 

"my_agent": {
  "name": "My Custom Agent",
  "icon": "🔥",
  "task_type": "creative",  // chat | creative | reason | code | private
  "system": "You are ... (your system prompt here)"
}

Save the file, restart the server (python3 goat_intel.py), and your agent appears on this page automatically. That's it.
