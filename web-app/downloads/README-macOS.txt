║              SUPER GOAT ROYALTY APP - macOS INSTALLATION GUIDE               ║
╚══════════════════════════════════════════════════════════════════════════════╝

⚠️  IMPORTANT: macOS Security Bypass Required

Since this app is not notarized by Apple, macOS will show a security warning.
Follow these steps to run the app:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 1: Right-Click Open (Recommended)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Download and extract the ZIP file
2. Move "Super GOAT Royalty.app" to your Applications folder
3. RIGHT-CLICK (or Control+Click) on the app
4. Select "Open" from the context menu
5. Click "Open" when prompted about the unidentified developer
6. The app will launch and be trusted for future opens

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 2: System Preferences
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Try to open the app normally (it will show an error)
2. Go to System Preferences → Privacy & Security
3. Scroll down to the Security section
4. Click "Open Anyway" next to the security warning
5. Enter your password if prompted
6. Click "Open" to confirm

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 3: Terminal Command (Advanced)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open Terminal (Cmd+Space, type "Terminal")
2. Run this command:

   xattr -cr /Applications/Super\ GOAT\ Royalty.app

3. Now you can open the app normally

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🍎 For Apple Silicon (M1/M2/M3) Users:

The app runs natively on both Intel and Apple Silicon Macs.
If you see "Rosetta 2" prompt, click "Install" to enable Intel app support.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
