Local Deployment Guide

Deploy GOAT Royalty App from your personal server with local drives and web access. Complete offline-capable system.

Prerequisites

Hardware

  • • 8TB Hard Drive (downloaded content)
  • • 16GB+ RAM
  • • Multi-core CPU (4 cores recommended)
  • • GPU with 8GB+ VRAM (for AI)

Software

  • • Node.js 18.x or later
  • • Docker & Docker Compose
  • • Nginx (optional)
  • • Git
1

Extract Downloaded Archive

Extract the complete archive to your local system

# Navigate to download location
cd /path/to/downloads/
# Extract archive
tar -xzf goat-royalty-complete.tar.gz
# Or on Windows:
tar -xzf goat-royalty-complete.tar.gz
2

Configure Environment

Set up your local environment configuration

# Navigate to extracted directory
cd goat-royalty-complete/
# Copy environment template
cp .env.example .env
# Edit .env with your settings
nano .env

Key Settings

  • • Set your local IP address
  • • Configure port numbers
  • • Set path to 8TB hard drive
  • • Configure AI model locations
3

Start with Docker

Launch all services using Docker Compose

# Start all services
docker-compose up -d
# Check status
docker-compose ps
# View logs
docker-compose logs -f
4

Access Your Local Instance

Open your browser and access the application

Local URL:
http://localhost:3000
Network Access (if configured):
http://YOUR-LOCAL-IP:3000

Troubleshooting

Services won't start?

Check if ports are already in use: netstat -tulpn

Can't access from other devices?

Check firewall settings and ensure the app is binding to 0.0.0.0 instead of localhost

AI models not loading?

Verify the AI model path in .env points to your 8TB hard drive
